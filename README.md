# Data Science Portfolio

A premium, single-page interactive personal portfolio website designed with a warm minimalist aesthetic (Charcoal & Amber). Built using semantic HTML5, modern CSS, and Vanilla JavaScript.

## Features

- **Warm Minimalist Aesthetic:** Sleek charcoal backgrounds, elegant amber/gold accents, and premium typography.
- **Fully Responsive:** Beautifully optimized for desktop, tablet, and mobile devices.
- **Interactive Elements:**
  - Dynamic typewriter effect on the hero section.
  - Interactive Project Filtering (All, Machine Learning, Deep Learning, Data Analytics).
  - Intersection Observer animations (smooth fade-in reveals on scroll).
  - Modern vertical experience timeline.
  - Active navigation highlight on scroll.
  - Interactive client-side validated Contact Form.

## File Structure

- `index.html` - Semantic HTML structure
- `style.css` - Custom styling using CSS variables and modern Grid/Flexbox layouts
- `script.js` - Dynamic interactions and animations
- `.gitignore` - Standard git exclusions

## Getting Started

To view the portfolio locally, simply open the `index.html` file in any web browser, or run a local development server.

If you have Node.js installed, you can run:
```bash
npx serve .
```

## Customization

1. **Modify Text:** Open `index.html` and update the personal details, projects list, and experience section.
2. **Change Colors:** Modify the CSS variables inside `style.css` under the `:root` pseudo-class.
